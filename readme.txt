The project has been built using Eclipse Mars Edition.

Here are a few basic instructions to set it up.

1) Create a workspace in Eclipse and import the project in it.
2) Use any app server like Tomcat etc to deploy the project.
3) The project uses Spring , JTA , Hibernate and is developed for a MySQL DB (SQLs used in the project may need a bit of tweaking in case of a different database).
4)  Please use the following url to test the project through a browser.
    http://localhost:8080/KabbadiProj/teamschedule/6
